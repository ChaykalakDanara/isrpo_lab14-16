## Описание проекта

Многостраничный сайт интернет-магазина электроники "TechStore" с адаптивной вёрсткой.

## Реализованные страницы

- **Главная** — приветственный баннер, популярные товары (3 карточки), преимущества (4 блока)
- **Каталог** — сетка из 9 карточек товаров
- **О нас** — информация о магазине, миссия, достижения (5+, 50K+, 1000+, 99%), команда (3 человека), 6 преимуществ

## Реализованные преимущества на странице "О нас"

1. ⭐ Оригинальная продукция
2. 💰 Выгодные цены
3. 🚚 Быстрая доставка
4. 🔧 Гарантия качества
5. ✅ Проверенное качество
6. 🏆 Награды и признание

## Технологии

- HTML5
- CSS3 (Flexbox, Grid, Media Queries, анимации, градиенты)
- Bootstrap 5
- Git/GitHub
